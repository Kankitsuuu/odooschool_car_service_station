Odoo School: Car Service

This module provides automation of the operation of a car service station

Installation

To install this module, you need to:

  • Go to **Apps** > **Apps** > **Main Apps**.
  • Search the module by name (os_car_service).
  • Click "Activate" button.

Note: If you are using the database in demo mode, the demo data will be loaded automatically.

Simple Usage

For simple use, you need to:

  • Go to **MainMenu** > **Car Service Management**.
  • Open **Clients**, **Workers* or **Services**.
  • Create new records via clicking "New" button.

Invoicing

After adding clients, workers, machines, services, create the provided services. After this you can create an invoice.
There are two ways to create an invoice:
1. Go to **Operations** > **New invoice**. Choose client and provided services. Click "Create" button.
2. Go to **Invoices** > "New". Choose client, provided services, also you can set "paid" status for invoice.

To print invoice, you need to:

  • Go to **Invoices**.
  • Pick one or more invoices.
  • Click **Print** > **Print invoice**
